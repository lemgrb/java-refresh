# Lambda

## Unsa napod ni?

- Kani kay bag-o bag-o ni sa Java 8 (Mga March 2014)
- e-treat ang **Functionality as method argument** o di ba kaha ang **Code as data**
- Para nis gitawag na 'Functional Interface'. Mga interfaces na usa ray abtract method. Pede nuon daghan na *default methods* o *static methods*.  

## Unsay 'blema?

- Kung simple ra kaayo imong anonymous class. i.e. usa ra ka method ang gi-implement kung nag `implements <Interface>`, ma-overkill ra kaayo ang Syntax nobra ka daghag codes.
  - Ang ato ra untang gusto buhaton kay mo-pasa og 'Functionality' as 'argument' sa ubang methods.
    - Example, `onClick()` unsay (pass a behavior) mahitabo?
